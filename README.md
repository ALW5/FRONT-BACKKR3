# PWA Заметки с Напоминаниями и Push-уведомлениями

Учебный проект, демонстрирующий комплексное применение прогрессивных веб-технологий: Service Worker, Web App Manifest, WebSocket, Push-уведомления и офлайн-доступ.

## Стек технологий

- **Бэкенд**: Node.js, Express.js, Socket.io, web-push
- **Фронтенд**: HTML5, CSS, JavaScript (ES6+)
- **PWA**: Service Worker, Web App Manifest, App Shell
- **Хранение данных**: localStorage
- **Сетевое взаимодействие**: WebSocket, Push-уведомления (VAPID)

## Функциональные возможности

### 📝 Управление заметками
- Создание, просмотр и удаление заметок
- Автоматическое сохранение в `localStorage`
- Работа в реальном времени через WebSocket

### 📱 PWA-возможности
- Установка приложения на устройство (Android/iOS/Desktop)
- Полная офлайн-работа (кэширование статики и динамики)
- Мгновенная загрузка благодаря архитектуре App Shell

### ⏰ Напоминания и Push
- Создание заметок с привязкой к дате и времени
- Push-уведомления даже при закрытой вкладке
- Кнопка "Отложить на 5 минут" в уведомлении

### 🔄 Режимы синхронизации
- Активные клиенты получают обновления мгновенно (WebSocket)
- Подписанные устройства — через Push-уведомления

Особенности реализации
VAPID-ключи обеспечивают безопасную идентификацию сервера при отправке push

Socket.io используется только для активных клиентов (меньше нагрузки на push-сервисы)

Стратегии кэширования гибко настроены под разные типы ресурсов

setTimeout на сервере — упрощённый вариант (в продакшене лучше использовать node-cron или очередь задач)
